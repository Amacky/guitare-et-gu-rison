Create Logo Showcase with Slick Slider

== Description ==

Display logo slideshow responsive slider/carousel on their website. Logo Showcase with Slick Slider help to display partners, clients or sponsor's Logo in your WordPress site quickly and easily. 

Using Logo Showcase with Slick Slider plugin creating a carousel slider of logos like client logo slider, partners logo slider, sponsor logo slider is super easy. 

View [FREE DEMO](http://demo.infornweb.com/slick-logo-showcase-demo/) for more details.

= Here is the shortcode example =

<code>[slick_logo_carousel id="XX"]</code>

= Features include: =
* Display Client logoshowcase in slider/carousel view.
* Display Unlimited Client logoshowcase.
* Add Link for image.
* Enable center mode (Shown in Demo)
* Display Logo including / excluding Title (Show or hide logo title)
* Multiple sliders can be shown.
* Slider sliding speed, autoplay Intervel, navigation, pagination, Slide columns for Logo slider, Number of Logos slides at a time.
* Created with Slick Slider.
* Display slider in Loop OR not

== Installation ==
1. Upload the 'logo-showcase-with-slick-slider' folder to the '/wp-content/plugins/' directory.
2. Activate the "logo-showcase-with-slick-slider" list plugin through the 'Plugins' menu in WordPress.
